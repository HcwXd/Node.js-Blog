# Blog

> [Demo](https://my-mean-blog.herokuapp.com/posts) - It's deployed on Heroku!



## Installation

```
$ git clone https://github.com/HcwXd/Blog.git
$ cd Blog
$ npm install 
```



## Usage

- Run it with online database MLab

```
npm run start
```

- Run it with local database Mongodb on port 27017

```
npm run test
```
